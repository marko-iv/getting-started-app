# Node.js Docker Tutorial

## Objective

Complete the Docker Node.js tutorial and run the application in a container.

## Steps Completed

1. Created the Node.js application.
2. Created the Dockerfile and compose.yaml files.
3. Built the Docker image using Docker Compose.
4. Started the Node.js and PostgreSQL containers.
5. Connected the application to the PostgreSQL database.
6. Created and retrieved a hero record using the API.

## Screenshots

* Build and startup process
* Server listening on port 3000
* POST request creating a hero
* GET request returning the hero

## Result

The application successfully ran in Docker and communicated with a PostgreSQL database.
